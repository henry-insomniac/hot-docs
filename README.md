# Hot Docs

开源、轻量、静态优先的 Docs + Blog 文档系统（v2.0 需求与架构已落盘）。

## 快速开始（开发）

```bash
pnpm install
pnpm dev
```

默认读取 `hot-docs.config.json`，内容目录为 `content/`。

## 构建与预览（静态产物）

```bash
pnpm site:build
pnpm site:preview
```

## 测试

```bash
pnpm test
```
