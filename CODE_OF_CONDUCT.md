# 行为准则（Code of Conduct）

本项目采用 Contributor Covenant v2.1。

## 我们的承诺

为了营造开放、友善与无骚扰的环境，我们承诺：无论年龄、体型、可见或不可见的残障、族裔、性别特征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教或性取向，我们都将参与并贡献一个让所有人都感到安全与受欢迎的社区。

## 我们的标准

有助于建立积极环境的行为包括（但不限于）：

- 以尊重、同理心与专业的方式沟通
- 接受建设性批评并聚焦问题本身
- 承担责任并在犯错后及时纠正
- 优先考虑对社区最有利的结果

不可接受的行为包括（但不限于）：

- 侮辱、骚扰、歧视或人身攻击
- 发布他人隐私信息
- 任何形式的性化言论或不当示好
- 破坏性行为（刷屏、钓鱼、恶意挑衅等）

## 执行责任

项目维护者有权并有责任澄清行为准则，并将对不可接受的行为采取适当且公平的纠正措施。

## 适用范围

本准则适用于所有项目空间（GitHub issue、PR、讨论区等），以及在代表项目时发生的互动。

## 处理流程

如需报告问题，请通过 GitHub issue/讨论区联系维护者，并提供尽可能详细的上下文。维护者将以保密、尊重的方式处理。

## 署名

本行为准则基于 Contributor Covenant（https://www.contributor-covenant.org/version/2/1/code_of_conduct.html）。
