# /milestone - 里程碑跟踪

## 描述
跟踪里程碑进度，生成状态报告，识别里程碑风险，协调跨模块依赖。

## 适用场景
- 周例会进度汇报
- 开源发布准备度检查
- 版本发布前验收
- 里程碑风险评估

## 工作流程
1. 读取 docs/todo.md 的里程碑任务
2. 统计完成度、进行中、待开始任务
3. 识别阻塞问题和风险
4. 生成可视化进度报告
5. 提供下一步行动建议

## 输入示例

### 示例 1：查看里程碑进度
```
/milestone M1
/milestone 查看 M1 Build 产物的完成情况
```

### 示例 2：评估发布准备度
```
/milestone 评估 v0.1.0 发布准备度
```

### 示例 3：里程碑对比
```
/milestone 对比 M0 和 M1 的进度
```
