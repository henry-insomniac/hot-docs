# /contributing - 贡献指南助手

## 描述
编写开源贡献指南、行为准则，帮助社区参与项目开发。

## 内容框架
1. 如何贡献（bug 报告/功能建议/代码贡献）
2. 开发环境（pnpm install/项目结构/调试）
3. 代码规范（分层原则/命名约定/类型要求）
4. 提交规范（Conventional Commits/PR 模板）
5. 插件开发（如何开发和发布第三方插件）
6. 行为准则（Code of Conduct）

## 工作流程
1. 分析项目架构与规范
2. 设计贡献流程
3. 编写开发指南
4. 补充示例与模板
5. 添加行为准则

## 输入示例
```
/contributing 编写 CONTRIBUTING.md
/contributing 编写插件贡献指南
/contributing 编写 CODE_OF_CONDUCT.md
```
