# /analyze - 代码库深度分析

## 描述
深度分析代码库结构、依赖关系、技术债务，生成可视化报告。

## 分析维度
- 模块依赖图（识别循环依赖/耦合度）
- 代码复杂度（圈复杂度/函数长度）
- 类型覆盖率（any 类型占比）
- 重复代码（可抽象的公共逻辑）
- 未使用代码（Dead code 检测）

## 工作流程
1. 扫描代码库结构
2. 分析依赖关系
3. 计算复杂度指标
4. 识别技术债务
5. 生成分析报告
6. 提供改进建议

## 输入示例
```
/analyze 分析 packages/core 的模块依赖
/analyze 识别代码库技术债务
/analyze 生成代码质量报告
```

## 分析工具
- dependency-cruiser（依赖分析）
- madge（循环依赖检测）
- ts-prune（未使用代码）
- eslint复杂度规则
